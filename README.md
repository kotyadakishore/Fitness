🏋️ Fitness Guidance Chatbot

An AI-powered Fitness Guidance Chatbot that helps users achieve their health goals by providing personalized workout plans, nutrition tips, and daily motivation.

Built using Python, Streamlit, and Groq API, this chatbot delivers real-time fitness advice through an interactive and user-friendly chat interface.

📌 Features
💬 Interactive AI Chat Interface
🏋️ Personalized Workout Plans
🥗 Nutrition & Diet Suggestions
🎯 Goal-Based Fitness Guidance (Weight Loss, Muscle Gain, etc.)
📅 Daily Fitness Tips & Motivation
⚡ Quick Options (Home Workout, Gym Plan, Diet Plan)
🔁 Chat Memory using Session State

🛠️ Tech Stack
Technology	Purpose
Python	Core Programming
Streamlit	Frontend UI
Groq API	AI Model (LLM)
dotenv	Environment Variable Management

🧠 How It Works
User enters a fitness-related query
Chat history is maintained using Streamlit session state
The query is sent to the Groq LLM API
The AI generates a personalized fitness response
Output is displayed in an interactive chat interface

🎯 Use Cases
Beginners starting their fitness journey
Users looking for home or gym workouts
Individuals aiming for weight loss or muscle gain
People seeking diet and nutrition guidance
Anyone needing daily fitness motivation
